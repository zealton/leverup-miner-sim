品质矿机 / 排放 · 市值 · 成本 模型

新设定：矿机是消耗品，分 A / B / C 三档品质（只发售 A，B / C 仅能通过 remint 抽取，总量固定）；每台矿机线性释放固定总量的 LV，交易量映射的 boost（1–3×）只加快释放速度、不改变总量，且每周衰减；钱包每刷 100 万美元交易量获得一次 remint 机会。左侧所有参数可调，页面顶部三张卡片实时回答：① LV 排放该控制在什么范围　② 什么市值下收益「实在 / 爆炸」　③ 协议的成本与损益。

01 / 认购
公平发售 A 品质
创世批 + 后续批次，售价 x MON。一个钱包可买多台。
02 / 释放
线性产出 LV
每台矿机内置固定 LV 总量，按秒线性释放，释放完矿机报废。
03 / BOOST
刷量加速释放
交易量绑定单台矿机 → boost 1–3×。只缩短周期、不加总量，每周衰减。
04 / REMINT
品质抽奖
每 $1M 交易量 = 1 次机会。按剩余配额加权随机，可能保持原品质；配额满则抽不到。
05 / 升级
进度继承
升级不重置时间：按当前进度比例，剩余时间释放新品质对应的剩余 LV。
