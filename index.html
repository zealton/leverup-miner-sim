品质矿机 / 排放 · 市值 · 成本 模型

新设定：矿机是消耗品，分 A / B / C 三档品质（只发售 A，B / C 仅能通过 remint 抽取，总量固定）；每台矿机线性释放固定总量的 LV；boost = 我的正 PnL ÷ 全体矿工的正 PnL 池（赚钱的人挖得快；亏损周只有下限速度；交易量本身不给 boost，只攒 remint 机会），映射到 1–3×，只加快释放、不改变总量，且开局一段时间后映射下限从 1 降为 0（完全不刷量 = 停止释放）；钱包每刷 100 万美元交易量获得一次 remint 机会。左侧所有参数可调，页面顶部三张卡片实时回答：① LV 排放该控制在什么范围　② 什么市值下收益「实在 / 爆炸」　③ 协议的成本与损益。

01 / 认购
公平发售 A 品质
创世批 + 后续批次，售价 x MON。一个钱包可买多台。
02 / 释放
线性产出 LV
每台矿机内置固定 LV 总量，按秒线性释放，释放完矿机报废。
03 / BOOST
盈利竞速
boost = 我的正 PnL（60h-EMA）÷ 全体矿工盈利池，映射 1–3×，每小时更新。EMA ≤ 0 吃下限；后期下限归 0。交易量不给 boost，只攒 remint。
04 / REMINT
品质抽奖
每 $1M 交易量 = 1 次机会。按剩余配额加权随机，可能保持原品质；配额满则抽不到。
05 / 升级
进度继承
升级不重置时间：按当前进度比例，剩余时间释放新品质对应的剩余 LV。
